About QEMU WebAssembly

QEMU is a powerful open-source emulator that has been compiled to WebAssembly to run in browsers. Unlike v86 which only supports 32-bit x86 CPUs, QEMU can emulate full 64-bit x86_64 systems.

64-bit CPU Capabilities

  • Full x86_64 instruction set support
  • Can run modern 64-bit Linux distributions
  • Support for larger memory addresses and modern applications
  • Better compatibility with contemporary software

Building Your Own QEMU-wasm Setup

To create a custom QEMU-wasm emulator with your own OS images, you need to:

  1. Clone the QEMU-wasm repository: git clone https://github.com/ktock/qemu-wasm.git
  2. Build QEMU with Emscripten (requires Docker):
cd qemu-wasm
docker build -t qemu-wasm .
# Follow the build instructions in the repository README

The build process generates:

  • qemu-system-x86_64.wasm - The WebAssembly binary
  • qemu-system-x86_64.js - The JavaScript loader
  • Supporting files for your OS image (kernel, initramfs, rootfs)

Comparison: v86 vs QEMU-wasm

Feature v86 (Original) QEMU-wasm (New)
CPU Architecture 32-bit x86 only 64-bit x86_64
Setup Complexity Simple (pre-built files) Complex (requires build)
Performance Fast Moderate (WebAssembly JIT)
OS Support 32-bit OSes only Modern 64-bit OSes
File Size ~2MB (v86.wasm) ~15-20MB (QEMU wasm)
